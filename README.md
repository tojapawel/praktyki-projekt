# praktyki-projekt - (hotel-finder)

autor: Kozicki Paweł

### uruchomienie

pierwsze uruchomienie:
```bash
cd api-server
npm install
npm start
```
```bash
cd hotel-finder
npm install
npm run dev
```

uruchonmienie dockera (na screen'ach):
```bash
cd api-server
npm start
```
```bash
cd hotel-finder
npm run dev
```
```bash
docker-compose up -d
```
