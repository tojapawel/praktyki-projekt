# praktyki-projekt - (hotel-finder)

autor: Kozicki Paweł

### uruchomienie

pierwsze uruchomienie:
```bash
cd api-server
npm install
```
```bash
cd hotel-finder
npm install
```

uruchomienie dockera (na screen'ach w podanej kolejności):

1. 
```bash
docker-compose up -d
```
2. 
```bash
cd api-server
npm start
```
3. 
```bash
cd hotel-finder
npm run dev
```
